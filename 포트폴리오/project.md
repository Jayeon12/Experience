### 💻 実装コードの一部（抜粋）

```java
public class Item {
    String name;
    int quantity;

    public Item(String name, int quantity) {
        this.name = name;
        this.quantity = quantity;
    }

    public void addStock(int amount) {
        this.quantity += amount;
    }

    public void removeStock(int amount) {
        if (this.quantity >= amount) {
            this.quantity -= amount;
        }
    }
}
```
